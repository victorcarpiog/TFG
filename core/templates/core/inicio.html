Bienvenidos al Test de Depresión

Por Victor Carpio

Este proyecto consiste en implementar el test de Beck para reconocer la depresión, consta de 21 preguntas que el usuario deberá de responder para conocer su resultado. No está validado por profesionales.




Para que el modelo funcione correctamente, evite responder con monosílabos.


Este test es una ayuda al diagnóstico, es preferible que busque ayuda profesional.
